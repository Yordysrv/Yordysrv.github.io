Que hacemos

Aqui algunas tareas que realiza un desarrollador web

Recopilacion de informacion

La recopilación de datos es el proceso de recogida de información mediante la observación y la experimentación. Los datos recogidos son una representación de datos y pueden estar en texto, números, imágenes o cualquier otro tipo de formato.

Codificacion

La codificación en una página web es el proceso de escribir código HTML, CSS y JavaScript para crear sitios web. El código fuente es el conjunto de datos, comandos y definiciones que se convierten en los elementos que componen la página web.

Mantanimiento

El mantenimiento web es el conjunto de tareas que se realizan para que un sitio web funcione correctamente y esté optimizado. El objetivo es que la página web transmita una imagen profesional de la empresa.

Revicion y lanzamiento

La revisión y el lanzamiento de una página web son etapas importantes en el proceso de creación de un sitio web.

Redacion y ensamblaje de contenido

La redacción de contenido es el proceso de planificar, redactar y editar contenido para tu web, normalmente con fines de marketing digital. Puede incluir escribir posts de blog, artículos, guiones para vídeos y podcasts, así como contenido para plataformas específicas, como tweetstorms en Twitter o textos en Reddit.

Diceño y prueva

El diseño y la prueba de una página web son procesos que permiten evaluar la funcionalidad y la experiencia del usuario. Diseño Diseñar teniendo en cuenta al público, Proporcionar una jerarquía visual, Que sea sencillo, Mostrar una llamada a la acción clara (CTA), Hacerlo accesible.
